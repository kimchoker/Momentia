## 📋 실행방법
1. 레포지토리 복제 후 의존성 설치
```
$ git clone [https://github.com/wjstjdus96/byhand.git](https://github.com/kimchoker/Momentia.git)
$ cd Monentia
$ pnpm install
```
2. 개발 서버 가동
```
$ pnpm run dev
```
3. 브라우저에서 실행
```
http://localhost:3000/
```
